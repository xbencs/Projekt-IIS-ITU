### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/xbencs/Projekt-IIS-ITU.git
   ```
3. Install composer packages
   ```sh
   composer install
   ```
4. Generate key
   ```sh
   php artisan key:generate
   ```
5. Create tables in database
   ```sh
   php artisane migrate
   ```
6. For testing start server on local machine
   ```sh
   php artisan serve
   ```
